Dist::Zilla::PluginBundle::ARJONES
==================================

This is the Dist::Zilla plugin bundle that ARJONES uses.

Also includes a default Pod::Weaver config.

Heavily based on https://github.com/rjbs/dist-zilla-pluginbundle-rjbs.
